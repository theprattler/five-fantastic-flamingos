# Work Day Scheduler

## Purpose
To provide a daily work task organizer broken into hour blocks that allows text-entered tasks to be saved and recalled in local storage.

## Built With
* HTML
* JavaScript
* CSS
* Bootstrap
* jQuery
* Moment.js
